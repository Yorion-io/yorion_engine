# Nepal Festival BS-RRULE Reference

BS-RRULE strings for Nepal's 53 major festivals. Every rule is validated against the official Nepali Panchanga almanac for BS 2079–2083 (208 tests in `festival_ground_truth`).

All rules use `X-SKIPADHIK=TRUE` and `X-TAKE=FIRST` unless noted. Dates are calculated at Kathmandu sunrise.
